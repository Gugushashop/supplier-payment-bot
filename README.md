# Telegram Bot для подсчёта оплат поставщикам

Автоматический бот для отслеживания и подсчёта платежей поставщикам на основе сообщений в Telegram.

## 🎯 Функциональность

✅ Автоматический парсинг заказов (ТТН + сумма)
✅ Подсчёт минусов (списания за отмены и доставку)
✅ Накопление данных до 17:00
✅ Отправка сводки в 17:00 в личку
✅ Трекинг новых реквизитов
✅ Экспорт в Excel
✅ Интеграция с Google Sheets (опционально)

## 📋 Требования

- Python 3.10+
- Telegram Bot Token
- Твой Telegram ID

## 🚀 Установка

### 1. Клонируем репо

```bash
git clone https://github.com/Gugushashop/supplier-payment-bot.git
cd supplier-payment-bot
```

### 2. Создаём виртуальное окружение

```bash
python -m venv venv
source venv/bin/activate  # Linux/Mac
# или
venv\Scripts\activate  # Windows
```

### 3. Устанавливаем зависимости

```bash
pip install -r requirements.txt
```

### 4. Настраиваем .env файл

```bash
cp .env.example .env
```

Отредактируй `.env` и добавь:
```
BOT_TOKEN=твой_токен_от_BotFather
ADMIN_ID=твой_telegram_id
```

### 5. Запускаем бота

```bash
python main.py
```

## 🔧 Настройка поставщиков

Отредактируй `config.py` и добавь твоих поставщиков:

```python
SUPPLIERS = {
    "gugusha_poliit": {
        "name": "Gugusha-Poliit",
        "chat_id": -1001234567890,  # Chat ID чата
        "requisites": [
            {"account": "4149123456789999", "currency": "USD"},
            {"account": "5167987654321111", "currency": "UAH"},
        ]
    },
    # ...
}
```

### Как узнать Chat ID?

1. Добавь бота в чат с поставщиком
2. Бот автоматически определит и выведет Chat ID
3. Добавь его в конфиг

## 📝 Команды бота

- `/start` - начало
- `/help` - справка
- `/summary` - сводка за сегодня
- `/balance` - баланс
- `/status` - статус бота

## 🐛 Решение проблем

### Бот не запускается
- Проверь `BOT_TOKEN` в `.env`
- Убедись что установлены все зависимости: `pip install -r requirements.txt`

### Не ловит сообщения
- Добавь бота в чаты поставщиков как участника
- Проверь `chat_id` в `config.py`

### Сводка не отправляется в 17:00
- Проверь что бот запущен
- Смотри логи: `python main.py`
